## Estructura

CINEPHILE cuenta con las siguientes 7 vistas:

- Index (Página Principal).
- Registro.
- Login.
- Recuperar Contraseña.
- Vista de Perfil.
- Detalle de pelicula seleccionada + Peliculas similares.
- Detalle de serie seleccionada + Series similares.
